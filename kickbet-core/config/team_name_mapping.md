## 映射规则说明

1. **FC/AFC后缀移除**: Football-Data.org使用完整名称如"Arsenal FC"，Odds-API.io常简化为"Arsenal"
2. **特殊字符简化**: "Mönchengladbach" → "Monchengladbach" (去掉ö)
3. **英文名转换**: "FC Bayern München" → "Bayern Munich"
4. **城市名替换**: "Juventus FC" → "Juventus Turin" (Odds使用城市名)
5. **历史球队**: Odds-API.io可能只包含当前赛季球队

## 使用方法

在代码中使用此映射表进行队名标准化：

```python
TEAM_NAME_MAPPING = {
    # Football-Data.org → Odds-API.io
    'Arsenal FC': 'Arsenal FC',
    'Aston Villa FC': 'Aston Villa',
    'Brighton & Hove Albion FC': 'Brighton & Hove Albion',
    # ... 更多映射
}

def normalize_team_name(fd_name: str, target_api: str = 'odds_api') -> str:
    """队名标准化"""
    if target_api == 'odds_api':
        return TEAM_NAME_MAPPING.get(fd_name, fd_name.replace(' FC', '').replace(' AFC', '')
    return fd_name
```
